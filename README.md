# Smart Parking System 

This project is a Smart Parking System developed using **C++, Node.js, HTML, CSS and JavaScript**. The aim of this project is to manage parking slots, track vehicles entering and exiting the parking area, and calculate revenue generated from parking.

The main parking logic is implemented in **C++ using Object Oriented Programming concepts**, while the **web interface** is built using HTML, CSS and JavaScript. A Node.js server is used to connect the frontend with the C++ program.

## Features

* Vehicle Parking Entry
* Vehicle Exit Management
* VIP and Emergency Parking
* Search Vehicle by Number
* View Available Parking Slots
* Parking History Records
* Admin Dashboard for Revenue Tracking

## Technologies Used

* C++
* Node.js (Express)
* HTML
* CSS
* JavaScript
  
## How the Project Works

1. The user interacts with the parking system through the web interface.
2. The frontend sends requests to the Node.js server.
3. The server runs the C++ program which contains the main parking logic.
4. The C++ program processes parking operations and stores data using file handling.
5. The result is sent back to the webpage and displayed to the user.


## Deployment

The project is deployed using **GitHub and Render**. The Node.js server runs the C++ program to process parking operations and display results on the web interface.
